FIREBASE SUDAH DIMASUKKAN

Project:
- projectId: softpro-11da5
- authDomain: softpro-11da5.firebaseapp.com

Cara tes:
1. Upload semua file ke GitHub Pages.
2. Buka aplikasi.
3. Masuk menu Pusat / Super Admin.
4. Buka Firebase Sync.
5. Klik Simpan Semua atau Simpan Tenant.
6. Buka Firebase Console > Firestore Database.
7. Cek collection:
   - wargahub_system/state
   - tenants/{tenantId}
   - tenants/{tenantId}/app/data
   - tenants/{tenantId}/residents
   - tenants/{tenantId}/reports
   - tenants/{tenantId}/services
   - tenants/{tenantId}/products

Penting:
Untuk testing, Firestore rules boleh dibuat longgar dulu.
Untuk produksi, wajib pakai Firebase Authentication dan security rules berbasis tenantId/role.
