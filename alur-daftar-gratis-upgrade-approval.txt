ALUR DAFTAR GRATIS + UPGRADE + APPROVAL

1. Pengurus/Admin daftar gratis.
   - Sistem membuat slot tenant otomatis.
   - Sistem membuat link admin dan kode undangan.
   - Status tenant: free_registered.
   - accessOpen: false.

2. Admin belum bisa mengundang warga.
   - Saat mau buka link warga / daftar warga, sistem meminta upgrade paket.

3. Admin pilih paket.
   - RT Mini, RT Plus, RW, Perumahan, Custom.
   - Admin mengirim bukti bayar.
   - Status tenant: pending_payment.
   - Request masuk ke Admin Pusat.

4. Admin Pusat approve.
   - Buka menu Approval Pembayaran.
   - Klik Approve & Buka Akses.
   - Status tenant: approved.
   - accessOpen: true.

5. Setelah approved:
   - Admin bisa membuat link warga.
   - Warga bisa daftar mandiri.
   - Admin bisa memberi mandat ke Admin RT/Bendahara/Satpam.
   - Admin RT bisa mengelola ruang RT masing-masing.

Catatan:
Versi demo ini masih memakai localStorage + Firebase Sync. Untuk produksi, approval harus tersimpan di Firestore dan diamankan dengan Firebase Auth role super_admin.
